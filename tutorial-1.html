OpenHAB UI Guide

You will now learn all necessary basics to get you going with openHAB.
Right now you are using the "Setup and Maintenance" interface, also called Paper UI in the community.

This tutorial and also this interface is strictly using new openHAB 2 technology.

Please be careful when reading user topics in our community, as those can reference older techniques. You want to avoid topics about:

  • Sitemaps
  • DSL Rules
  • .thing, .item files

What openHAB can do for you

openHAB is a mature home automation platform that is protocol agnostic, meaning it can connect to nearly any home automation hardware on the market today.

Usually you just need to install an additional piece of software via this web interface, use a feature detection of your hardwares capabilities and you can already control your external solution.

Via timers and rules you start automating like turning your lights on when your phone is in WiFi range, but only if it is already later than 6 pm. Only your imagination is the limit.

openHAB also and supports DIY hardware. We speak about this later on in the tutorial.

About this interface

Use this interface to setup new devices, configure existing ones, create automation rules and timers.

You do not control your devices via this interface. Download the Android App, iOS App, HaBot or use HabPanel instead.
