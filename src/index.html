Extend openHAB

A fresh installation has a limited functionality only.

Find the section "Add-ons" in the top navigation bar and extend your openHAB installation. Choose from over 300 addons.

An addon that enables you to use a physical device is called a binding.

Concepts - Bindings and Things

A binding provides Things. A Thing can have Channels.

Concepts - Channels

A Channel has a unique ID, a specific Type (Text, Number, Percentage, ...), an optional unit (like Temperature in degrees celsius) and a State.

Concepts - Items

Channels are not used directly to control devices or visualize device states.

Instead you link a Channel to one or more Items.

This abstraction seems unnecessary at first glance, but allows for some interesting concepts and complex scenarios.

Habpanel

When I have set up my smart home, how do I control it via openHAB?

This is where you need to leave Paper UI (this is really setup and maintenance only). We provide an Android App, an iOS App and also a web-based solution for all kind of devices called Habpanel.

Make sure you have installed Habpanel in the Add-ons section.
