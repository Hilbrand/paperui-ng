Habpanel

When I have set up my smart home, how do I control it via openHAB?

This is where you need to leave Paper UI (this is really setup and maintenance only). We provide an Android App, an iOS App and also a web-based solution for all kind of devices called Habpanel.

Make sure you have installed Habpanel in the Add-ons section.
